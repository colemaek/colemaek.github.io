What is Colemäk?

Colemäk is a mod of Colemak-DH for Finnish Colemak users (preferably with an ortholinear keyboard).

Experimental

Ä and Ö

Ä and Ö will maybe get switched with the default comma and dot. Time will tell.

Just FYI, I'm getting this keyboard for training Colemäk.

ZSA Planck EZ

You don't need to change the default QWERTY layout for this to work, if I understood correctly.
Colemäk will be stored in the keyboard's memory.
