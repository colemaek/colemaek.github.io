What is Colemäk?

Colemäk is a mod of Colemak-DH for Finnish Colemak users (preferably with an ortholinear keyboard).

Experimental

Ä and Ö

Ä and Ö will maybe get switched with the default comma and dot. Time will tell.

Personal journey

I recommend the following process for learning Colemäk:

Colemak Academy

In Colemak Academy, start from Level 1 with the settings "Colemak DHm".
When you get over 10wpm and 95% accuracy in a level, go up one level.

MonkeyType

After accomplishing the Academy, head over to MonkeyType and start practicing by using the "Colemak DH matrix" keymap emulator. I like to do 10 words per session only, so that I can see the progress happening easily.
